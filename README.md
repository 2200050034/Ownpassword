# 🔐 Java Password Check System

A simple Java program that allows users to **set a password** and **verify it with limited login attempts**.

---

## 🚀 Features

* 👤 Set your own password
* 🔑 Login with password verification
* 🔁 Maximum 3 login attempts
* ❌ Access denied after failed attempts
* ✅ Access granted on correct password

---

## 🛠️ Technologies Used

* Java
* Scanner (for user input)

---

## ▶️ How to Run

1. Compile the program:

```bash
javac PasswordCheck.java
```

2. Run the program:

```bash
java PasswordCheck
```

---

## 💻 Example Output
